# Interactive Hover Parallax Effect

<!-- 
HOW TO USE THIS HOVER PARALLAX EFFECT:
1. The effect works by switching between different SVG images on hover
2. Each SVG has the same elements but with slightly different positions
3. When you hover over each section, it appears to have depth and movement
4. This is compatible with GitHub as it uses only standard markdown features
-->
